# Sales Performance & Customer Insights Dashboard

## Overview

This project analyzes sales performance and customer behavior using Power BI. The dashboard provides insights into revenue trends, regional sales performance, and customer segmentation.

## Tools Used

* Power BI
* DAX
* Data Modeling
* SQL (for data preparation)
* Excel

## Key Features

* Multi-page Power BI dashboard
* Customer segmentation analysis
* Regional sales performance comparison
* Revenue trend analysis
* Drill-through to individual customer transactions

## Key Metrics

* Revenue Growth
* Customer Lifetime Value (CLV)
* Churn Rate
* Month-over-Month Revenue Variance
* Rolling 3-Month Average Sales
